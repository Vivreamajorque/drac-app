5 modes d'apprentissage

Chaque mode adapte l'interface, les textes, les feedbacks et le rythme. Pas un style de façade — une vraie différence pédagogique validée scientifiquement.

🐉
Standard

Parcours complet, interface équilibrée. Pour tous ceux qui ne se retrouvent dans aucun autre profil.

10 mots / session Feedback normal
🌀
TDAH

Sessions courtes (5 mots), récompenses toutes les 2 bonnes réponses, confettis à chaque bonne réponse, messages ultra-enthousiastes.

5 mots / session Toast boost Confettis fréquents
📖
Dys

Police Arial avec espacement agrandi, hauteur de ligne 1.9, priorité à l'écoute (synthèse vocale), feedback calme et rassurant.

Arial, letter-spacing Line-height 1.9 Oral prioritaire

HPI / Avancé

Interface sobre Inter, contexte étymologique approfondi, deux boutons d'écoute séparés CA / ES, feedback intellectuel.

Étymologie Interface sobre Double écoute

Installer Drac sur ton téléphone

Drac est une PWA — une vraie app installable sans passer par l'App Store. Icône sur l'écran d'accueil, fonctionne hors ligne, aucun compte requis.

1

Ouvre drac-app.vercel.app dans Safari (iPhone) ou Chrome (Android)

2

Appuie sur le bouton Partager (iOS) ou les 3 points (Android)

3

"Ajouter à l'écran d'accueil" → L'icône 🐉 apparaît
